bool bpg::Puzzle::add_ship ( const Ship & ship)

Adds a ship to the puzzle board.

This function tries to add a ship to the puzzle board. It first checks if the ship overlaps with another ship or margin on the board. If it does, the function returns false and the ship is not added. If the ship does not overlap, the function iterates over the cells of the ship, fills them with a ship cell state, fills the margins of these cells, and returns true.

Parameters
shipThe ship to be added to the board.
Returns
True if the ship was added to the board, false otherwise.
Note
The *this is a pointer to the object itself
My idea is to return this to the methods in Generator so it can perform the tracking to failed attempts and try again.

std::string bpg::Puzzle::to_string_matrix ( ) const

Converts the current state of the puzzle to a string representation.

This method generates a string representation of the current state of the puzzle. The string includes the number of rows and columns, and a matrix representation of the puzzle. Each cell in the matrix is represented by a symbol that indicates whether it is water or part of a ship. The symbols used are defined in the ship_symbols array.

Returns
A string representation of the puzzle.
