bool bpg::Generator::add_unique_board_to_hash ( const Puzzle & board)

Adds a unique board to the hash set of unique boards.

This function takes a Puzzle object, converts it to a string representation, and checks if it already exists in the hash set of unique boards. If the board already exists in the hash set, the function returns false. If the board does not exist in the hash set, it is added and the function returns true.

Parameters
boardThe Puzzle object to be added to the hash set of unique boards.
Returns
A boolean indicating whether the board was successfully added to the hash set (true) or if it already existed in the hash set (false).

std::list< Puzzle > bpg::Generator::generate ( const RunningOpt & run_opt)

=== Main Methods

Generates a list of unique Puzzle objects.

This function generates a specified number of unique Puzzle objects and adds them to a list. Each Puzzle object represents a board configuration. The function ensures that each board configuration is unique by checking if it already exists in a hash set of unique boards. If a generated board configuration already exists in the hash set, it is discarded and the function tries again.

Parameters
run_optThe running options, which include the number of puzzles to generate.
Returns
A list of unique Puzzle objects.

Puzzle bpg::Generator::generate_single_board ( const RunningOpt & run_opt)

=== Main methods

Generates a single board for the Battleship puzzle.

This function generates a single board for the Battleship puzzle. It first generates the possible positions matrix if it has not been generated yet. Then, it creates a list of ships and adds the ships to the list. It shuffles the possible positions for each ship. Finally, it tries to add each ship to the board. If a ship cannot be added, it generates a new board.

Parameters
run_optThe running options for the board generation.
Returns
The generated board.
