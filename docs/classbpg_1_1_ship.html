◆ contains()

bool bpg::Ship::contains ( Cell cell) const

Checks if a ship contains a given cell.

This function checks if a given cell is contained within the ship. The function takes into account the ship's orientation. If the ship's orientation is horizontal (H), it checks if the cell's row is the same as the ship's row, and if the cell's column is within the range of the ship's column and the ship's column plus its length. If the ship's orientation is vertical (V), it checks if the cell's column is the same as the ship's column, and if the cell's row is within the range of the ship's row and the ship's row plus its length.

Returns
True if the ship contains the cell, false otherwise.

bool bpg::Ship::is_end ( Cell cell) const

Checks if a cell is the end of the ship.

This function checks if a given cell is the end (last position) of the ship. If the ship's orientation is horizontal (H), it checks if the cell's row is the same as the ship's row, and if the cell's column is the same as the ship's column plus its length minus one. If the ship's orientation is vertical (V), it checks if the cell's column is the same as the ship's column, and if the cell's row is the same as the ship's row plus its length minus one. If the ship's orientation is undefined (U), it checks if the cell's row and column are the same as the ship's row and column.

std::string bpg::Ship::to_string ( ) const

Converts the ship's details to a string.

This function converts the ship's type, position, and orientation to a string. The ship's type is represented by a character ('B', 'D', 'C', or 'S'). The ship's position is represented by its row and column coordinates. The ship's orientation is represented by a character, unless the orientation is 'U' (up).
